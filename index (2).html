CSE 160 — Computer Graphics

Lauren Liu — WebGL & Three.js Projects

Three projects built over one course, progressing from raw WebGL fundamentals to a full Three.js scene graph: a voxel world engine, a Phong lighting system, and an interactive 3D space scene.

Voxel World Engine

WebGL · Asg3

First-person, Minecraft-style 3D world with procedural terrain, real-time block editing, and an interactive NPC mini-game — built without a game engine.

View live demo →

Phong Lighting & Shading System

WebGL · GLSL · Asg4

Full Phong lighting pipeline with dynamic point lights, spotlights, and per-fragment normal visualization, plus real-time controls for light position, color, and intensity.

View live demo →

Floating Space World

Three.js · Asg5

Interactive 3D scene combining imported GLB models (Phoenix, UFO) with custom procedural geometry, multiple light types, a day/night cycle, and a live UI control panel.

View live demo →
